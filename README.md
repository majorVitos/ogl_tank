# ogl_tank

OpenGL tank model it is just an example of OpenGL low skill. The T32 is a prototype https://en.wikipedia.org/wiki/T-34, made by text without any modeling programs like blender or 3dsmax
Модель танка в OpenGL. Прототип Т34. Модель сделана в текстовом редакторе без применения программ типа blender или 3dsmax.

## Getting Started
Just copy and run.
Keys: A - toogle animate, R - toogle rotate, T - toogle textures, C - toogle mode, F - toogle fog, L - toogle light, P - toogle frustum
